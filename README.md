# BMW 325i Badge — Baseline v0.1

First printable silhouette for the connected 325i emblem.

## Dimensions
- Width: 184.15 mm
- Height: 31.75 mm
- Maximum thickness: 4.7625 mm
- Recessed rear bridge thickness: 1.35 mm

## Included
- Printable STL
- Editable SVG outline
- Preview PNG
- CadQuery script for STEP export

## Generate STEP
Run this in your CadQuery environment:

```powershell
python src\build_badge.py
```

This baseline focuses on silhouette and proportions. Crowned faces, drafted sides,
edge bevels, and mounting features are reserved for later revisions.
